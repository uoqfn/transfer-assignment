# A Small Git Adventure

One night, a developer was trying to fix a bug that refused to disappear. 
Luckily, their project was tracked with git, so nothing was ever truly lost.
But then something went wrong. A dangerous command was executed, and files disappeared.
Thankfully, git history made it possible to travel back in time and recover the work.
From that day on, the developer respected branches, commits, and pull requests.
Git was no longer just a tool — it was a safety net.
